# Journal Entry Preparation

Best practices, standard entry types, documentation requirements, and review workflows for journal entry preparation.

> **Important**: All entries should be reviewed by qualified financial professionals before posting.

## Standard Accrual Types

### AP Accruals
- Debit: Expense (or capitalize if asset-qualifying), Credit: Accrued liabilities
- Sources: Open POs with confirmed receipts, contracts with unbilled services, recurring vendor arrangements, unprocessed expense reports
- Auto-reverse in following period; document estimation basis; track actual vs accrual

### Fixed Asset Depreciation
- Debit: Depreciation/amortization expense (by dept), Credit: Accumulated depreciation
- Methods: Straight-line (most common), Declining balance, Units of production
- Verify new additions, check for disposals/impairments, ensure book vs tax consistency

### Prepaid Amortization
- Debit: Expense, Credit: Prepaid expense
- Common: Insurance, software licenses, rent, maintenance contracts, event deposits
- Maintain schedule with start/end dates; check for cancelled contracts requiring accelerated amortization

### Payroll Accruals
- Salary accrual (working days in period vs pay period), Bonus accrual (plan terms), Benefits accrual, Payroll tax accrual
- Include employer-side taxes and benefits (FICA, FUTA, health, 401k match)

### Revenue Recognition
- Follow ASC 606 five-step framework
- Identify performance obligations, determine transaction price, allocate, recognize as/when satisfied
- Maintain contract-level detail for audit support

## Supporting Documentation Requirements

Every entry needs:
1. Clear description/memo
2. Calculation support
3. Source document references
4. Period identification
5. Preparer identification
6. Approval evidence
7. Reversal indicator

## Approval Matrix

| Entry Type | Threshold | Approver |
|-----------|-----------|----------|
| Standard recurring | Any | Accounting manager |
| Non-recurring manual | < $50K | Accounting manager |
| Non-recurring manual | $50K-$250K | Controller |
| Non-recurring manual | > $250K | CFO / VP Finance |
| Top-side / consolidation | Any | Controller or above |
| Out-of-period adjustments | Any | Controller or above |

## Common Errors to Check

1. Unbalanced entries
2. Wrong period
3. Wrong sign (debit as credit)
4. Duplicate entries
5. Wrong account
6. Missing reversal on accruals
7. Stale accruals not updated
8. Round-number estimates without calculations
9. Incorrect FX rates
10. Missing intercompany elimination
