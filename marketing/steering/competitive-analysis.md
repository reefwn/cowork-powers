# Competitive Analysis (Marketing)

Frameworks for researching competitors, comparing positioning, and identifying market opportunities from a marketing perspective.

## Research Methodology

### Sources
**Primary**: Website, blog, social media, product demos, webinars, press releases, job postings.
**Secondary**: Review sites (G2, Capterra), analyst reports, news coverage, social listening, SEO tools, financial filings, community forums.

### Process
Set scope → gather data → organize by competitor and dimension → analyze patterns → compare to your position → synthesize insights → date-stamp everything.

### Cadence
Deep analysis quarterly. Monitoring monthly. Real-time alerts ongoing.

## Messaging Comparison

### Messaging Matrix
Compare: tagline, core value prop, primary audience, key differentiator, tone/voice, proof points, category framing, primary CTA.

### Narrative Analysis
For each competitor identify: Villain (what they position against), Hero (customer? product?), Transformation (before/after), Stakes (what if you don't act?).

### Messaging Assessment
Evaluate: Clarity, Differentiation, Proof, Consistency, Resonance.

## Content Gap Analysis

Map content by topic across competitors. Compare content type coverage (blog, case studies, ebooks, webinars, podcast, video, tools, templates).

Identify: Topics they cover that you don't, topics you cover that they don't, format gaps, audience segment gaps, SEO keyword gaps.

## Positioning Strategy

### Positioning Statement
> For [target audience], [product] is the [category] that [key benefit] because [reason to believe].

### Positioning Map
Plot on 2x2 matrix: Price vs Capability, Ease vs Power, SMB vs Enterprise, Point Solution vs Platform.

### Category Strategy Options
Create new category, reframe existing category, win existing category, or niche within category.

## Battlecard Creation

Structure: Quick overview → Their pitch → Strengths (be honest) → Weaknesses → Our differentiators → Objection handling → Landmines to set → Landmines to defuse → Win/loss themes.

Update quarterly minimum, immediately after major competitor announcements, and with win/loss feedback from sales.
