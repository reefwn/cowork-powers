# Content Creation

Guidelines and frameworks for creating effective marketing content across channels.

## Content Type Templates

### Blog Post
Headline (benefit-driven, <60 chars) → Introduction with hook (100-150 words) → 3-5 body sections with H2 subheadings → Conclusion with CTA (75-100 words) → Meta description (<160 chars).

### Social Media
Hook (first line grabs attention) → Body (2-4 points or short narrative) → CTA → Hashtags (3-5).

Platform specifics: LinkedIn (professional, paragraph breaks, ~1300 chars), Twitter/X (concise, punchy), Instagram (visual-first, storytelling), Facebook (conversational, questions).

### Email Newsletter
Subject line (<50 chars) → Preview text → Header/hero → 2-3 scannable body sections → Primary CTA → Footer.

### Landing Page
Headline (<10 words) → Subheadline → Hero with CTA → 3-4 value propositions → Social proof → FAQ/objection handling → Final CTA.

### Press Release
Headline → Dateline → Lead paragraph (who/what/when/where/why) → Body → Boilerplate → Media contact.

### Case Study
Title ("[Customer] achieves [result] with [product]") → Snapshot → Challenge → Solution → Results with metrics → Quote → CTA.

## Writing Best Practices

- Blog: 8th-grade reading level, short paragraphs, subheadings every 200-300 words, active voice
- Email: personalize, one primary CTA, scannable, mobile-first
- Web: lead with benefits, "you" language, minimize jargon, reduce friction near CTAs

## SEO Fundamentals

One primary keyword + 2-3 secondary per piece. Primary keyword in: headline, first paragraph, one subheading, meta description, URL slug. Don't keyword-stuff.

On-page checklist: title tag <60 chars, meta description <160 chars, clean URL slug, one H1, descriptive H2/H3, image alt text, 2-3 internal links, 1-2 external links.

## Headline Formulas

- "How to [result] [without obstacle]"
- "[Number] [adjective] ways to [result]"
- "Why [common belief] is wrong (and what to do instead)"
- "The [adjective] guide to [topic]"

## Hook Formulas

Surprising statistic, contrarian statement, question, scenario, bold claim, story opening.

## CTA Best Practices

Action verbs ("Get", "Start", "Try"). Be specific about what happens next. Reduce risk ("No credit card required"). One primary CTA per page/email. Place above the fold on landing pages, after establishing value in emails.
