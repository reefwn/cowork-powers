# Performance Report Workflow

Generate a marketing performance report with metrics, trends, and optimization recommendations.

## When to Use
When the user wants a marketing report, performance analysis, campaign results, or metrics summary.

## Inputs

1. **Report type**: Campaign, Channel (email/social/paid/SEO), Content performance, Overall marketing, Custom
2. **Time period**: last week/month/quarter or custom range
3. **Data source**: product analytics (if connected) or ask user to provide metrics
4. **Comparison period** (optional): prior period or YoY
5. **Stakeholder audience** (optional): executive summary vs detailed analyst view

## Output Structure

1. **Executive Summary**: 2-3 sentences, headline metric with trend, one win, one concern
2. **Key Metrics Dashboard**: Table with this period, prior period, change, target, status (on track / at risk / off track)
3. **Trend Analysis**: Performance over time, inflection points, seasonality, benchmark comparison
4. **What Worked**: Top 3-5 wins with data, hypotheses, how to replicate
5. **What Needs Improvement**: Bottom 3-5 with data, hypotheses, recommended fixes
6. **Insights and Observations**: Non-obvious patterns, audience behavior, external factors
7. **Recommendations**: Per recommendation — what, why (linked to data), expected impact, effort, priority. Include 2x2 impact/effort matrix.
8. **Next Period Focus**: Top 3 priorities, tests to run, metric targets

See **performance-analytics** steering for metrics definitions and benchmarks by channel.
