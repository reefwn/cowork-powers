# Email Sequence Workflow

Design and draft complete email sequences with full copy, timing, branching logic, and performance benchmarks.

## When to Use
When the user wants to create an email sequence, drip campaign, nurture flow, or onboarding series.

## Inputs

1. **Sequence type** (required): Onboarding, Lead nurture, Re-engagement, Product launch, Event follow-up, Upgrade/upsell, Win-back, Educational drip
2. **Goal** (required): what the sequence should achieve
3. **Audience** (required): who receives it, their stage, segmentation details
4. **Number of emails** (optional): recommend based on type if not specified
5. **Timing/cadence** (optional)
6. **Brand voice**: auto-apply if configured
7. **Additional context** (optional): offers, CTAs, content assets, features to highlight

## Process

### 1. Sequence Strategy
Define: narrative arc, journey mapping (awareness → decision → activation), escalation logic, success/exit definition.

### 2. Per Email
- Subject line (2-3 options), preview text (40-90 chars)
- Email purpose (one sentence)
- Full body copy (short paragraphs, scannable, personalization tokens)
- Primary CTA (button text + destination)
- Timing (days after trigger/previous email)
- Segment/condition notes

### 3. Sequence Logic
Branching conditions, exit conditions (conversion = exit), re-entry rules, suppression rules.

### 4. Performance Benchmarks

| Metric | Onboarding | Lead Nurture | Re-engagement | Win-back |
|--------|-----------|--------------|---------------|----------|
| Open rate | 50-70% | 20-30% | 15-25% | 15-20% |
| CTR | 10-20% | 3-7% | 2-5% | 2-4% |
| Conversion | 15-30% | 2-5% | 3-8% | 1-3% |
| Unsubscribe | <0.5% | <0.5% | 1-2% | 1-3% |

## Sequence Type Templates

- **Onboarding** (5-7 emails, 14-21 days): Welcome → Quick win → Core feature → Advanced feature → Social proof → Check-in → Upgrade
- **Lead Nurture** (4-6, 3-4 weeks): Educational value → Pain point → Solution + proof → Social proof → Soft CTA → Direct CTA
- **Re-engagement** (3-4, 10-14 days): "We miss you" → Value reminder → Incentive → Last chance
- **Win-back** (3-5, 30 days): Check-in → What's new → Special offer → Feedback request → Final goodbye
- **Product Launch** (4-6, 2-3 weeks): Teaser → Launch → Feature spotlight → Social proof → Limited offer → Reminder
- **Upgrade/Upsell** (3-5, 2-3 weeks): Usage milestone → Feature gap → Upgrade benefits → Incentive → Plan comparison

## Output

1. **Sequence Overview Table**: #, subject line, purpose, timing, CTA, condition
2. **Full Email Drafts**: Each with all elements
3. **Sequence Flow Diagram**: Text-based with branching and exit points
4. **A/B Test Suggestions**: 2-3 recommended tests
5. **Metrics to Track**: Per-email and sequence-level

## Follow Up
Offer to: revise copy, add branching paths, create variation for different segment, draft A/B variants, build companion sequence.
