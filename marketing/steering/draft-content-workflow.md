# Draft Content Workflow

Generate marketing content drafts tailored to content type, audience, and brand voice.

## When to Use
When the user wants to draft, write, or create marketing content.

## Inputs

1. **Content type** (required): Blog post, Social media (specify platform), Email newsletter, Landing page, Press release, Case study
2. **Topic** (required)
3. **Target audience** (required)
4. **Key messages** (required): 2-4 main takeaways
5. **Tone** (optional if brand voice configured)
6. **Length** (optional): target word count or format constraint

## Brand Voice
Apply configured brand voice automatically. If not configured, ask or use neutral professional tone.

## Content Generation

Follow the templates in the **content-creation** steering for each type:

- **Blog post**: 2-3 headline options, hook intro, 3-5 sections with subheadings, conclusion with CTA, SEO recommendations
- **Social media**: Platform-appropriate format, hook first line, hashtags, CTA, emoji per brand
- **Email newsletter**: 2-3 subject lines, preview text, body sections, CTA button text
- **Landing page**: Headline/subheadline, hero copy, 3-4 value props, social proof placeholder, CTAs, FAQ suggestions, meta title/description
- **Press release**: Standard format with headline, dateline, lead, quotes placeholder, boilerplate
- **Case study**: Result-focused title, challenge → solution → results, quote placeholder, CTA

## SEO (for web content)
Suggest primary keyword, recommend placement, suggest meta description, note image alt text opportunities.

## Follow Up
Offer to: revise sections, adjust tone, create variation for different channel.
