# SEO Audit Workflow

Audit a website's SEO health, research keyword opportunities, identify content gaps, and benchmark against competitors.

## When to Use
When the user wants an SEO audit, keyword research, content gap analysis, technical SEO check, or competitor SEO comparison.

## Inputs

1. **URL or domain** (required)
2. **Audit type**: Full site audit (default), Keyword research, Content gap analysis, Technical SEO check, Competitor SEO comparison
3. **Target keywords** (optional)
4. **Competitors** (optional): if not provided and needed, identify 2-3 via web search

## Process

### 1. Keyword Research
If SEO tools connected: pull keyword data, volume, difficulty, rankings automatically.
If not: use web search. Note that connecting Ahrefs or Semrush provides more precise data.

Assess: primary keywords, secondary keywords, volume signals, difficulty, long-tail opportunities, question-based keywords, intent classification.

### 2. On-Page SEO Audit
For key pages evaluate: title tags, meta descriptions, H1 tags, H2/H3 structure, keyword usage, internal linking, image alt text, URL structure.

### 3. Content Gap Analysis
Competitor topic coverage, content freshness (12+ months stale), thin content, missing content types, funnel gaps, topic cluster opportunities.

### 4. Technical SEO Checklist
Page speed, mobile-friendliness, structured data opportunities, crawlability (robots.txt, sitemap, canonicals), broken links, HTTPS, Core Web Vitals signals, indexation.

### 5. Competitor SEO Comparison
Keyword overlap and gaps, domain authority signals, content depth, backlink profile, SERP feature ownership, technical advantages.

## Output

### Executive Summary
3-5 sentences: biggest strength, top 3 priorities, overall assessment.

### Keyword Opportunity Table
| Keyword | Est. Difficulty | Opportunity Score | Current Ranking | Intent | Recommended Content Type |
15-25 keywords sorted by opportunity score.

### On-Page Issues Table
| Page | Issue | Severity | Recommended Fix |
Severity: Critical, High, Medium, Low.

### Content Gap Recommendations
Per gap: topic, why it matters, recommended format, priority, estimated effort.

### Technical SEO Checklist
| Check | Status | Details |
Status: Pass, Fail, Warning.

### Competitor Comparison Summary
| Dimension | Your Site | Competitor A | Competitor B | Winner |

### Prioritized Action Plan
**Quick Wins (this week)**: <2 hours, immediate impact.
**Strategic Investments (this quarter)**: longer effort, long-term growth.

## Follow Up
Offer to: draft content briefs for top keywords, create optimized title tags/meta descriptions, build content calendar from gap analysis, dive deeper, or run for different competitor/domain.
